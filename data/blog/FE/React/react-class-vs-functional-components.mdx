### 클래스형 컴포넌트 vs 함수형 컴포넌트
##### 클래스형 컴포넌트
1. state기능을 사용할 수 있다.
2. lifecycle 기능을 사용할 수 있다.
3. 임의 메서드를 정의할 수 있다.
4. render 함수가 꼭 있어야 한다.

##### 함수형 컴포넌트
1. 클래스형 컴포넌트보다 선언하기 편함
2. 메모리 자원을 클래스형 컴포넌트보다 덜 사용한다.
3. 프로젝트 빌드 & 배포 시 결과물 파일 크기가 더 작다.
4. state와 라이프사이클 API 사용이 불가하다 --> v16.8이후 Hooks ㄱ도입으로 해결됨.
