#### 문제 설명
    정수가 담긴 리스트 num_list가 주어집니다. 
    num_list의 홀수만 순서대로 이어 붙인 수와 짝수만 순서대로 이어 붙인 수의 합을 return하도록 solution 함수를 완성해주세요.
#### 제한사항
````
2 ≤ num_list의 길이 ≤ 10
1 ≤ num_list의 원소 ≤ 9
num_list에는 적어도 한 개씩의 짝수와 홀수가 있습니다.
````
#### 입출력 예
| num_list        |	result|
|-----------------|-----|
| [3, 4, 5, 2, 1] |	393|
| [5, 7, 8, 3]    |	581|
---
#### 풀이
````
function solution(num_list) {
    let odd = num_list.filter((item,idx)=>item%2!==0).reduce((acc,cur)=>acc.toString()+cur.toString())
    let even = num_list.filter((item,idx)=>item%2===0).reduce((acc,cur)=>acc.toString()+cur.toString())
    
    return parseInt(odd)+parseInt(even)
}
````
---
#### 알게된 점
다른 사람의 풀이를 보니

`return +numA + +numB;`
라는 코드가 있었는데,

String 형 변수 앞에 +를 붙여주면 암묵적으로 타입 변환이 일어나서 숫자형으로 바꿔준다고 한다 우와!
