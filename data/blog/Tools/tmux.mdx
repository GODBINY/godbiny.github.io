# tmux
- 터미널 멀티플렉
- 단일 터미널 창을 여러 세션으로 나눠 사용할 수 있도록 하는 도구.
- 여러 세션 생성,전환,삭제 가능 / 화면 분할 가능 / ** 네트워크 연결 끊겨도 세션과 작업이 유지됨 **

# tmux 기본개념
1. session
- tmux가 관리하는 가상의 터미널 집합.
- 하나의 세션은 여러 windows를 가질 수 있음.
- 각각의 세션은 독립적으로 이어져있음.
- ssh 연결이 끊겨도 세션은 살아있으며, 다시 접속&이어작업 가능함.

2. window
- 세션 안에서 생성하는 새로운 작업 공간.

3. pane
- 윈도우 내부를 여러 개의 사각형 공간으로 쪼개서 각각 독립적인 터미널로 쓸 수 있게 해주는 기능.

# tmux 주요 명령어
````bash
# 세션 관련 명령어
# 새로운 세션 생성:
tmux new -s [세션이름] # 명령으로 이름을 지정해 세션을 만듭니다.

# 세션 목록 보기:
tmux ls 

# 세션 접속(attach):
tmux attach -t [세션이름] 

# 세션 분리(detach):
Ctrl+b를 누른 후 d로 현재 세션을 백그라운드로 전환.
 
# 세션 종료:
세션에서 exit을 입력하거나, 외부에서 tmux kill-session -t [세션이름]으로 세션을 종료.
 
# 윈도우(Window) 명령어
# 새 윈도우 생성:
Ctrl+b를 누른 후 c로 새로운 윈도우를 생성.
 
# 윈도우 이동:
Ctrl+b + 숫자로 해당 윈도우로 바로 이동.
 
# 윈도우 이름 변경:
Ctrl+b + ,로 윈도우 이름 변경.
 
# 윈도우 종료:
Ctrl+b + & 또는 패널에서 exit으로 윈도우를 종료.
 
# 윈도우 리스트 보기:
Ctrl+b + w로 전체 윈도우 목록을 볼 수 있음.
 
# 패널(Pane) 명령어
# 화면 분할(세로):
Ctrl+b + %로 패널을 좌우로 분할.
 
# 화면 분할(가로):
Ctrl+b + \" 로 위아래로 분할.
 
# 패널 사이 이동:
Ctrl+b + 방향키로 이동.
 
# 패널 삭제:
Ctrl+b + x로 패널을 종료. 확인창이 뜨면 y 입력.
 
# 패널 크기 조절:
Ctrl+b + : 입력 후 resize-pane -L/R/U/D [숫자]로 크기 변경.
 
# 기타 유용한 명령
# 복사 모드 진입 및 화면 스크롤:
Ctrl+b + [로 복사/스크롤 모드 진입, 방향키로 이동, q로 종료.
 
# 붙여넣기:
복사 후, Ctrl+b + ]로 붙여넣기 수행.
 
# 단축키 목록 보기:
Ctrl+b + ?로 전체 단축키를 조회.
````
