#### 문제 설명
    정수 n과 문자열 control이 주어집니다. control은 "w", "a", "s", "d"의 4개의 문자로 이루어져 있으며, control의 앞에서부터 순서대로 문자에 따라 n의 값을 바꿉니다.
    "w" : n이 1 커집니다.
    "s" : n이 1 작아집니다.
    "d" : n이 10 커집니다.
    "a" : n이 10 작아집니다.
    위 규칙에 따라 n을 바꿨을 때 가장 마지막에 나오는 n의 값을 return 하는 solution 함수를 완성해 주세요.
#### 제한사항
````
-100,000 ≤ n ≤ 100,000
1 ≤ control의 길이 ≤ 100,000
control은 알파벳 소문자 "w", "a", "s", "d"로 이루어진 문자열입니다.
````
#### 입출력 예
입출력 #1

|n	|control|	result|
|----|----|----|
|0|	"wsdawsdassw"	|-1|
---
#### 풀이
````
function solution(n, control) {
    control.split('').map((item)=>{
        switch(item){
            case 'w':
                n+=1
                break;
            case 's':
                n-=1
                break;
            case 'd':
                n+=10
                break;
            case 'a':
                n-=10
                break;
                }
    })
    return n;
}
````
````
function solution(n, control) {
    const move = {w:1,s:-1,d:10,a:-10}
    return control.split('').reduce((acc,cur)=>acc+move[cur],n)
}
````
````
function solution(n, control) {
    const move = {
        w: (n) => n+1,
        s: (n) => n-1,
        d: (n) => n+10,
        a: (n) => n-10
    }
    return [...control].reduce((acc,cur) => move[cur](acc) ,n)
}
````

#### 알게된 점
object에 함수를 넣어서 사용할 수 있다는 것을 알게됐다.

그리고 object 에 접근할 때에는 [] 를 사용하여 접근.
object 내에 선언한 함수에 접근하려면 () 를 사용해야하니까

`move[cur](acc)`

이렇게 접근하는것.

그리고 나는 string을 array로 바꿀 때 String.split('')으로 했었는데,

`[...String]` 이렇게 하면 array로 변경 가능하다는 것도 알게됐다.
