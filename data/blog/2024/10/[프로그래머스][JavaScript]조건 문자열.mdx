#### 문제 설명
문자열에 따라 다음과 같이 두 수의 크기를 비교하려고 합니다.

두 수가 n과 m이라면

&gt;, = : n &gt;= m

&lt;, = : n &lt;= m

&gt;, ! : n &gt; m

&lt;, ! : n &lt; m

두 문자열 ineq와 eq가 주어집니다. ineq는 "&lt;"와 "&gt;"중 하나고, eq는 "="와 "!"중 하나입니다. 그리고 두 정수 n과 m이 주어질 때, 
n과 m이 ineq와 eq의 조건에 맞으면 1을 아니면 0을 return하도록 solution 함수를 완성해주세요. 

#### 제한사항
    1 ≤ n, m ≤ 100
#### 입출력 예
입출력 #1

| ineq | eq | n  | m  | result |
|------|----|----|----|--------|
| "&lt;"  | "=" | 20 | 50 | 1      |
| "&gt;"  | "!" | 41 | 78 | 0      |


---
#### 풀이
````
function solution(ineq, eq, n, m) {
    var answer = '';
    if(ineq === '<'){
        if(eq === '=') {
            answer = (n<=m);
        } else {
            answer = (n<m);
        }
    } else if(ineq === '>'){
        if(eq === '=') {
            answer = (n>=m);
        } else {
            answer = (n>m);
        }
    }
    return answer ? 1 : 0;
}
````
#### 다른사람 코드 와.. 이렇게 푸네
````
const operations = {
  '>=': (n, m) => n >= m,
  '<=': (n, m) => n <= m,
  '>!': (n, m) => n > m,
  '<!': (n, m) => n < m,
};

function solution(ineq, eq, n, m) {
  const op = operations[ineq + eq];
  return Number(op(n, m));
}
````
